Addons are modular extensions that bolt extra functionality onto the Dimensions plugin without bloating the core. Each addon ships as its own `.jar` file, so you only load the features you actually need — keeping your server lean and giving the community room to contribute new ideas.

## Installing an Addon

To install any addon, place its `.jar` file inside the `./plugins/Dimensions/Addons/` folder on your server, then restart your server. The plugin automatically discovers every jar in that directory on startup.

You can download all free addons bundled together from the [DimensionsFreeAddons pack on SpigotMC](https://www.spigotmc.org/resources/dimensionsfreeaddons.97040/).
