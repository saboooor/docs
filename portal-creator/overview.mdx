The Portal Creator is a free browser-based tool that lets you design and configure Dimensions portals visually. Instead of hand-editing YAML, you interact with a live portal preview and download a ready-to-use config file when you're done. This page covers Portal Creator version 0.1.

Open the tool at **[https://astaspasta.alwaysdata.net/](https://astaspasta.alwaysdata.net/)** — no account or installation required.

## Interface layout

When you open the Portal Creator, you'll see three main areas:

- **Portal preview (center)** — A visual representation of your portal with clickable circles overlaid on it. Each circle corresponds to a different part of the portal you can configure.
- **Clickable circles on the portal** — Three interactive circles let you edit the frame block (top-left of the portal), the inside block (center of the portal), and the lighter item (lighter circle). Clicking any of them opens a configuration menu for that element.
- **Options box (left side)** — A panel to the left of the portal preview where you can set additional portal options and any addon-specific settings you have installed.

<Note>
  The Portal Creator runs entirely in your browser. Nothing is saved server-side — make sure to download your config file before closing the tab.
</Note>
