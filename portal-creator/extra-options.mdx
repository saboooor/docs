The Portal Creator's left-side panel sits to the left of the portal preview and gives you access to settings that go beyond the frame, inside block, and lighter. This is where you configure additional portal behavior options, as well as any addon-specific settings for the addons you have installed on your server.

## What the panel contains

The left-side options box is split into two categories:

- **Additional portal options** — Settings that apply to all portals, such as general behavior flags that would otherwise require you to open and edit the portal's YAML file by hand.
- **Addon-specific settings** — If you have any Dimensions addons installed, their configurable options appear here automatically. Each addon surfaces its own section in the panel so you can set values before you download the config.

<Info>
  The exact options visible in this panel depend on which addons you have installed. If you haven't installed any addons, only the core portal options appear.
</Info>

<Tip>
  Configure everything in the left-side panel **before** downloading your config file. Filling in these options now means you won't need to open the downloaded `.yml` file and edit it manually afterwards.
</Tip>

<Note>
  Every setting in this panel maps directly to a field in the portal's YAML configuration file. If you need to understand what a particular option does in depth, open the downloaded `.yml` file in a text editor — the key names match what you see in the panel.
</Note>
